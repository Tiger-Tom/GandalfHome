Ubuntu Autologin:
1: sudo systemctl edit getty@tty1.service
2: Add the lines:
[Service]
ExecStart=
ExecStart=-/sbin/agetts --noissue --autologin <USERNAME> %I $TERM
Type=idle
3: Save & close the file

Ubuntu Autostart:
1: How to run program on user login (default path is "/home/<USERNAME>/Gandalf", AKA "~/Gandalf", assuming that you have the command "python3")
2: You can either do 2a or 2b. They should have the same result. 2b is usually easier.
2a: Type "echo 'cd /home/<USERNAME>/Gandalf/ && python3 <PATH>/VoiceControl.py' >>/home/<USERNAME>/.bashrc"
2b: Add the line "cd /home/<USERNAME>/Gandalf/ && python3 <PATH>/VoiceControl.py" to "/home/<USERNAME>/.bashrc"
